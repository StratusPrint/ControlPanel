Welcome to your new StratusPrint Control Panel!

Getting Started

In order to get started, just simply add a HUB. That's it! Once the HUB is added, printer and environment updates will automatically begin populating in this control panel. You will then be able to manage any printer that is connected to the hub, including any new printers that might be connected in the future.

A status overview of each printer managed by the selected HUB will be displayed here.

In addition to an overview of each printer, graphs generated using data from any installed sensors will also be displayed here.

For additional help including access to installation instructions and tips on how to use this control panel, refer to https://docs.stratusprint.com/.
